   ✓ Charge base-layout.jsp
   ✓ <jsp:include page="../includes/head.jsp" />

   ✓ <jsp:include page="../includes/topbar.jsp" />
   ✓ <jsp:include page="${view}" /> ← INJECTE citoyen/liste.jsp
   ✓ <jsp:include page="../includes/footer.jsp" />

📞 POINTS DE CONTACT POUR DEBUGGING
================================================================================

Si 404 sur ressources CSS/JS:
→ Vérifier chemin dans base-layout.jsp: ${pageContext.request.contextPath}
→ Vérifier chemins dans head.jsp

Si page ne se charge pas:
→ Vérifier mapping Servlet @WebServlet("/path")
→ Vérifier chemin view dans setAttribute()
→ Vérifier forward dans RequestDispatcher

Si layouts ne s'assemblent pas:
→ Vérifier chemins relatifs dans base-layout.jsp
→ Vérifier noms des fichiers JSP (sensible à la casse!)

Si modales ne s'ouvrent pas:
→ Vérifier app-core.js chargé (F12 Network)
→ Vérifier openModal('citoyen') appelé
→ Vérifier id="modal-citoyen" présent dans footer.jsp
