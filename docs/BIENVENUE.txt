## 🚀 DÉMARRAGE RAPIDE (5 MINUTES)

### Étape 1: Compiler
```bash
mvn clean compile -DskipTests
```

### Étape 2: Générer le WAR
```bash
mvn clean package -DskipTests
```

### Étape 3: Déployer sur Tomcat
```bash
cp target/mon-premier-projet-1.0-SNAPSHOT.war $CATALINA_HOME/webapps/
$CATALINA_HOME/bin/startup.sh
```

### Étape 4: Accéder
```
http://localhost:8080/mon-premier-projet/
```

## 🔗 STRUCTURE DE NAVIGATION

```
Utilisateur accède:  http://localhost:8080/mon-premier-projet/
                     ↓
                GET /dashboard
                     ↓
        DashboardServlet.doGet()
                     ↓
     request.setAttribute("view", 
       "dashboard/index.jsp")
                     ↓
     forward("/WEB-INF/jsp/layouts/base-layout.jsp")
                     ↓
    base-layout.jsp ASSEMBLE:
  ┌─ head.jsp (scripts, CSS)
  ├─ sidebar.jsp (navigation)
  ├─ topbar.jsp (search, thème)
  ├─ ${view} ← dashboard/index.jsp (INJECTÉ)
  └─ footer.jsp (modales, scripts)
                     ↓
    PAGE COMPLÈTE AVEC LAYOUT
```

## 💡 CONCEPTS CLÉ DÉMONTRÉS

✓ **Composite View Pattern** - Fragments réutilisables + injection dynamique
✓ **MVC Architecture** - Séparation Modèle/Vue/Contrôleur
✓ **Servlet Technology** - @WebServlet, @WebFilter annotations
✓ **JSP Includes** - Réutilisabilité des composants
✓ **Context Path Handling** - ${pageContext.request.contextPath}
✓ **Dynamic View Injection** - ${view} en JSP
✓ **HTTP Navigation** - Vraies URLs au lieu de SPA
✓ **Resource Organization** - CSS/JS centralisés
